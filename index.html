Strings


            "Hello World"
            # => "Hello World

Strings


            "Hello World" + ", we're learning to program today"
            # => "Hello World, we're learning to program today"

This is called concatenation. We have used the + operator to join two strings together

Strings


            name = "Jamie"
            "Hello #{name}"
            # => "Hello Jamie

This is called Interpolation. This is a way to embed a variable value or expression inside of a string

Writing a Script

In a plain text file, you can write ruby!


            # crappy_calculator.rb
            puts 2 + 2

Running a Script

In the terminal, navigate to the file and run it with the ruby interpreter


            $ cd ~/Desktop
            $ ruby crappy_calculator.rb

cd means 'Change Directory'

The tilde stands for your Home Directory. On mac OS this is the folder that holds your pictures, videos and desktop folders

So this means go to the Desktop folder in the home directory and run ruby with my script

Interacting with the user in a script
